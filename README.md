Grade sense is a robust full-stack web application designed to empower students to take control of their academic journey. Built with the MERN (MongoDB, Express.js, React, Node.js) stack, this platform goes beyond basic GPA tracking. It provides powerful estimation tools to help you set future goals and offers personalized suggestions to guide you towards achieving your desired academic standing.

Whether you're planning your next semester, aiming for a specific GPA, or simply want to visualize your academic progress, Academic Compass offers the insights and tools you need to succeed.
